# 找到值为 0 的位置（题解）

## 问题描述

存在一个长度为 2n 的隐藏数组 a，其中：
- 整数 1 到 n 各出现 **恰好一次**
- 剩余 n 个位置都是 **0**

每次查询可以询问两个位置 i、j，交互器返回 1 表示 a[i] = a[j]，返回 0 表示 a[i] ≠ a[j]。

要求在不超过 n + 1 次查询内找到一个位置 k，使得 a[k] = 0。

## 算法

```python
def query(i, j):
    print(f"? {i + 1} {j + 1}")
    ans = int(input())
    return ans


def solve():
    n = int(input())

    # 第一步：检查相邻的奇偶位置对
    for i in range(2, 2 * n, 2):
        if query(i, i + 1) == 1:
            print(f"! {i + 1}")
            return

    # 第二步：检查 a[1] 是否为 0
    if query(0, 2) == 1 or query(0, 3) == 1:
        print("! 1")
    else:
        print("! 2")
```

## 正确性证明

### 关键性质

由于 1 到 n 每个数字恰好出现一次，**两个不同位置的值相等，当且仅当这两个位置都是 0**。

### 第一步的正确性

遍历 i = 2, 4, 6, ..., 2n-2，检查 (i, i+1) 即相邻的奇偶位置对。

- 如果 query(i, i+1) == 1，说明 a[i] = a[i+1]
- 根据关键性质，这意味着一对相邻位置都是 0
- 找到答案，算法结束

### 第二步的正确性

如果第一步没有找到答案，说明**没有相邻的 0**。此时 0 必定分散在数组中。

考虑位置 1（索引 0）：
- 如果 query(1, 3) == 1 或 query(1, 4) == 1，说明 a[1] = 0（与另一个 0 相等）
- 否则 a[1] ≠ 0，由于每个非零值只出现一次，a[1] 只能是 1~n 中的某个数

此时唯一可能为 0 的位置就是 1 或 2（因为 0 不可能与 1 相邻）。选择其中一个即可。

### 查询次数分析

- 第一步最多检查 n/2 个相邻对
- 第二步最多 2 次查询
- 总计 ≤ n/2 + 2 ≤ n + 1（当 n ≥ 2 时）

满足题目要求。

## 复杂度

- 时间：O(n) 次查询
- 空间：O(1)
