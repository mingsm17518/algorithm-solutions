# B. Array（题解）

## 问题描述

给定长度为 n 的数组 a，对于每个索引 i，找出满足以下条件的索引 j 的最大数量：
- `j > i`
- 存在整数 `k` 使得 `|a[i] - k| > |a[j] - k|`

## 条件理解

不等式 `|a[i] - k| > |a[j] - k|` 的含义是：
- 整数 `k` **距离 a[j] 比距离 a[i] 更近**

几何上，k 落在 a[i] 和 a[j] 中点偏向 a[j] 的一侧。

### 两种情况

1. **当 a[j] > a[i]**：选择非常大的 k（如 10^18）。由于 k 在很远的右边，有 |a[i] - k| > |a[j] - k|。

2. **当 a[j] < a[i]**：选择非常小的 k（如 -10^18）。由于 k 在很远的左边，有 |a[i] - k| > |a[j] - k|。

## 关键洞察

对于每个 i，我们只能选择**一个** k 值。目标是找到一个 k，使得满足条件的 j 尽可能多。

### 策略

- **方案 1**：选择非常大的 k。这对所有 a[j] > a[i] 的 j 都有效。
- **方案 2**：选择非常小的 k。这对所有 a[j] < a[i] 的 j 都有效。

索引 i 的答案就是：
```
max(右侧大于 a[i] 的元素数量, 右侧小于 a[i] 的元素数量)
```

## 算法

对于每个索引 i：
1. 统计右边有多少个元素大于 a[i]
2. 统计右边有多少个元素小于 a[i]
3. 输出 max(大于的数量, 小于的数量)

## 正确性证明

我们证明算法对每个索引 i 输出正确答案。

### 引理 1

对于任意 j > i 且 a[j] > a[i]，存在整数 k 使得 |a[i] - k| > |a[j] - k|。具体地，任意足够大的 k 都可以。

**证明**：如果 a[j] > a[i]，选择 k > (a[i] + a[j]) / 2，则：
- |a[i] - k| = k - a[i]
- |a[j] - k| = k - a[j]

由于 k - a[i] > k - a[j]，有 |a[i] - k| > |a[j] - k|。∎

### 引理 2

对于任意 j > i 且 a[j] < a[i]，存在整数 k 使得 |a[i] - k| > |a[j] - k|。具体地，任意足够小的 k 都可以。

**证明**：如果 a[j] < a[i]，选择 k < (a[i] + a[j]) / 2，则：
- |a[i] - k| = a[i] - k
- |a[j] - k| = a[j] - k

由于 a[i] - k > a[j] - k，有 |a[i] - k| > |a[j] - k|。∎

### 引理 3

固定 i，如果我们选择非常大的 k，条件对所有 a[j] > a[i] 的 j 成立，但对 a[j] < a[i] 的 j 不成立。

**证明**：直接由引理 1 和引理 2 可得。∎

### 定理

对于每个索引 i，算法输出的值是满足 |a[i] - k| > |a[j] - k|（对某个整数 k） 的 j 的最大可能数量。

**证明**：由引理 3，只有两种最优策略：
1. 选择非常大的 k → 对所有 a[j] > a[i] 有效
2. 选择非常小的 k → 对所有 a[j] < a[i] 有效

没有其他 k 值可以同时覆盖两种类型（它们要求 k 在数组的相反两侧）。因此最大值是 max(count_greater, count_smaller)，这正是算法计算的。∎

## 复杂度分析

每个测试用例：
- 两层嵌套循环：O(n²) 次比较
- 由于 n ≤ 5000 且所有测试用例 n 之和 ≤ 5000，最坏情况是 25,000,000 次操作，可以接受。

额外空间：O(1)。

## 参考实现（Python 3）

```python
t = int(input())

for _ in range(t):
    n = int(input())
    a = list(map(int, input().split()))
    for i in range(n):
        l = 0  # 右侧 a[j] < a[i] 的数量
        r = 0  # 右侧 a[j] > a[i] 的数量
        for j in range(i + 1, n):
            if a[j] < a[i]:
                l += 1
            if a[j] > a[i]:
                r += 1
        print(max(l, r), end=' ')
        if i == n - 1:
            print()
```

## 样例解释

### 测试用例 2：a = [105, -105]

- 对于 i = 1（0-indexed）：a[1] = 105
  - 右侧元素：-105（小于 105）
  - l = 1，r = 0 → 答案 = 1
- 对于 i = 2（0-indexed）：a[2] = -105
  - 右侧没有元素
  - 答案 = 0

输出：1 0 ✓
