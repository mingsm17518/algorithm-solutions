# 前缀和

## 子数组之和 是某数的倍数

同余 的两个下标之间的子数组 和为目标值的倍数

输出子数组的最长长度：[求最长的长度](./02_Subsequences_Summing.cpp)
记录每个余数首次出现的下标，再次出现时，与首次出现的下标的差值为目标子数组的长度，与最长值比较，记录大的数为最长值

输出子数组的个数：[求数组个数](./05_Subarray_Divisibility.cpp)
记录每个余数出现的次数，最后对每个余数出现的次数 求两两组合的个数

输出子数组的个数:
[数组中全为正数](./Subarray_Sums_I.cpp)
数组中无负数，可以用滑动窗口Sliding_Window求解
